## How to run

To start vite:

```javascript
npm run dev
```

I am using the Laravel Sail dev environment.
To use too, learn it from [Laravel Sail](https://laravel.com/docs/10.x/sail).

To start docker containers:

```bash
./vendor/bin/sail up
```
In detached mode:

```bash
./vendor/bin/sail up -d
```
